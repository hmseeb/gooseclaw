**BotInstance.start()**:
```python
def start(self):
    if self.running:
        return
    _session_manager.load(self.channel_key)
    register_notification_handler(self.channel_key, self._make_notify_handler())
    self.generate_pair_code()
    # register slash commands with Telegram API
    self._register_commands()
    self.running = True
    self._thread = threading.Thread(target=self._poll_loop, daemon=True)
    self._thread.start()
    print(f"[telegram:{self.name}] started")
```

**BotInstance.stop()**:
```python
def stop(self):
    self.running = False
    if self._thread:
        self._thread.join(timeout=5)
        self._thread = None
```

**BotInstance._register_commands()**: Same as current start_telegram_gateway's setMyCommands block, but uses self.token.

**BotInstance._make_notify_handler()**: Returns a closure that uses self.token and get_paired_chat_ids(platform=self.channel_key) to send notifications.
