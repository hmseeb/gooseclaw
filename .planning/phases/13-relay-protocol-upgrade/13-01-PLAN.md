<task type="auto">

  <action>
Add the following functions near the existing WS relay code (after _StreamBuffer, around line 5098).

**1. _parse_sse_events(response)**:
Generator that reads http.client HTTPResponse line-by-line. Lines starting with "data: " have the suffix parsed as JSON and yielded. Invalid JSON is skipped. Empty/non-data lines ignored. Returns when readline() returns empty bytes.

**2. _build_content_blocks(user_text, inbound_msg=None)**:
Build content block array for ChatRequest. If user_text has content, add {"type":"text","text":user_text}. If inbound_msg has media, iterate mc.to_content_block() and add non-None results. If no blocks at all, add {"type":"text","text":""} as fallback. Returns list of dicts.

**3. _extract_response_content(content_array)**:
Parse assistant message content array. Filter type=="text" blocks into text_parts (join with newline). Filter type=="image" blocks into media_blocks. For type=="toolResponse", check tool_result.value.content for nested text/image blocks. Skip thinking, reasoning, systemNotification. Returns (text_str, media_blocks_list).

**4. _do_rest_relay(user_text, session_id, content_blocks=None, sock_ref=None)**:
POST to goosed /reply, parse SSE response. Build ChatRequest JSON with session_id, user_message (role=user, created=timestamp, content=content_blocks or [text block], metadata). Auth via Basic base64(user:token). Use http.client.HTTPConnection("127.0.0.1", GOOSE_WEB_PORT, timeout=300). Store conn in sock_ref[0] for cancellation. Parse SSE events: Message -> extract text/media via _extract_response_content, Error -> return error, Finish -> break. Returns (text, error, media_blocks). Handle socket.timeout, ConnectionError, generic Exception like _do_ws_relay.

**5. _do_rest_relay_streaming(user_text, session_id, flush_cb, verbosity="balanced", content_blocks=None, sock_ref=None, flush_interval=4.0)**:
Same as _do_rest_relay but uses _StreamBuffer for incremental text delivery. Message events with text -> buf.append. toolRequest content -> flush buf, emit "[Using tool...]" via flush_cb. thinking content -> emit italic if verbose. Error -> return. Finish -> break. Returns (text, error, media_blocks).

Do NOT modify _relay_to_goose_web or any callers yet (that's Plan 02).
