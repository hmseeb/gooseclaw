_WATCHER_TICK_SECONDS = 30

_watcher_engine_tick():
- Get snapshot of enabled feed watchers under lock
- For each: check if now - last_check_ts >= poll_seconds
- If due: set last_check_ts = now, spawn daemon thread calling _check_feed_watcher
- Wrap in try/except, log errors

_watcher_engine_loop():
- Set _watcher_engine_running = True
- Print "[watchers] engine started"
- While _watcher_engine_running: call _watcher_engine_tick(), then sleep 5s x 6 iterations (30s total, checking _watcher_engine_running each iteration)

start_watcher_engine():
- If no watchers loaded (or _watcher_engine_running already True), skip
- Start daemon thread targeting _watcher_engine_loop
- Print "[watchers] engine started with N watcher(s)"

stop_watcher_engine():
- Set _watcher_engine_running = False
