Add WebSocket protocol functions to gateway.py at module scope, after the logging section (~line 140) and before the GatewayHandler class. Add a clear section header comment.

**Constants:**
```python
# ── websocket protocol (RFC 6455) ───────────────────────────────────────────

WS_MAGIC = "258EAFA5-E914-47DA-95CA-C5AB0DC85B11"
WS_OP_TEXT = 0x1
WS_OP_BINARY = 0x2
WS_OP_CLOSE = 0x8
WS_OP_PING = 0x9
WS_OP_PONG = 0xA
```

**Functions (in order):**

1. `ws_accept_key(client_key)` - Compute Sec-WebSocket-Accept from Sec-WebSocket-Key. SHA-1 hash of (key.strip() + WS_MAGIC), base64 encoded.

2. `_ws_recv_exact(sock, n)` - Read exactly n bytes from socket. Loop until all bytes received. Raise ConnectionError if socket closes mid-read.

3. `ws_recv_frame(sock)` - Read one WebSocket frame. Returns `(opcode, payload_bytes)` or `(None, b"")` on connection close.
   - Read 2-byte header
   - Extract FIN, opcode, masked flag, length
   - Handle all 3 length encodings: 7-bit (<126), 16-bit (==126, struct.unpack(">H")), 64-bit (==127, struct.unpack(">Q"))
   - Read 4-byte mask key if masked
   - Read payload using _ws_recv_exact
   - Unmask payload if masked: `bytes(b ^ mask_key[i % 4] for i, b in enumerate(payload))`
   - Wrap entire function in try/except (ConnectionError, OSError) returning (None, b"")

4. `ws_send_frame(sock, opcode, payload, mask=False)` - Send one WebSocket frame.
   - Build frame header: FIN=1 | opcode
   - Length encoding: <126 = 1 byte, <65536 = 126 + 2 bytes big-endian, else = 127 + 8 bytes big-endian
   - If mask: set mask bit, generate 4-byte key with os.urandom(4), XOR payload
   - Use sock.sendall()

5. `ws_send_ping(sock, mask=False)` - Send WebSocket ping frame. Just calls ws_send_frame(sock, WS_OP_PING, b"", mask=mask).

6. `ws_send_close(sock, code=1000, reason="", mask=False)` - Send close frame. Payload = struct.pack(">H", code) + reason.encode("utf-8"). Wrap in try/except (OSError, BrokenPipeError) to handle dead sockets gracefully.

7. `ws_start_ping_loop(sock, interval=25, mask=False)` - Start a daemon thread that sends WebSocket ping every `interval` seconds. Returns the Thread object. Thread exits when socket send raises OSError/BrokenPipeError.

**All stdlib imports are already present in gateway.py** (hashlib, base64, struct, socket, ssl, threading, os, time). No new imports needed.

**Do NOT add** any do_GET modifications, handler methods, or route detection in this task. That's Plan 02.
