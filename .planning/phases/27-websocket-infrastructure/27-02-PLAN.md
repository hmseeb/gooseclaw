**2. Connection tracking helper functions:**

- `_ws_register(conn_id, sock, ping_thread)` - Add connection to _ws_connections under lock. If at cap, close oldest connection first (send close frame, remove from dict).
- `_ws_unregister(conn_id)` - Remove connection from _ws_connections under lock. No error if not found.
- `_ws_active_count()` - Return len(_ws_connections) under lock.

**3. Outbound WebSocket client function (add at module scope):**

`ws_client_connect(host, path, query_params=None)` - Opens outbound WSS connection:
- Create TLS context with `ssl.create_default_context()`
- Open TCP connection with `socket.create_connection((host, 443), timeout=10)`
- Wrap with TLS: `ctx.wrap_socket(raw_sock, server_hostname=host)`
- Build WebSocket upgrade request (GET with Upgrade, Connection, Sec-WebSocket-Key, Sec-WebSocket-Version headers)
- Build URL from path + optional query_params using `urllib.parse.urlencode`
- Send handshake, read response until `\r\n\r\n`
- Verify status line contains "101"
- Verify Sec-WebSocket-Accept header matches expected value using `ws_accept_key()`
- Return the ssl-wrapped socket
- On failure: close socket, raise ConnectionError with descriptive message

**4. handle_voice_ws() method in GatewayHandler class:**

Add this method to GatewayHandler:

```python
def handle_voice_ws(self):
    """Upgrade HTTP to WebSocket and enter echo loop (Phase 27 scaffold)."""
```

Steps:
1. Validate Upgrade header is "websocket" (case-insensitive). If not, send_error(400, "Expected WebSocket upgrade"), return.
2. Get Sec-WebSocket-Key header. If missing, send_error(400, "Missing Sec-WebSocket-Key"), return.
3. Validate Origin header: allow if Origin matches `self.headers.get("Host")` or is empty (wscat/CLI tools). Log warning for mismatched origins but don't reject (Railway domain matching is complex).
4. Compute accept key using `ws_accept_key()`.
5. Send 101 Switching Protocols with headers: Upgrade: websocket, Connection: Upgrade, Sec-WebSocket-Accept: {accept}.
6. Call `self.end_headers()` then `self.wfile.flush()` (flush buffered data before switching to raw socket).
7. Set `self.close_connection = True` to prevent BaseHTTPRequestHandler from reading more HTTP.
8. Get raw socket: `sock = self.request`
9. Set socket timeout: `sock.settimeout(60)` (ping loop will keep connection alive, timeout catches dead connections)
10. Generate connection ID with `uuid.uuid4().hex[:8]`
11. Start ping loop: `ping_thread = ws_start_ping_loop(sock, interval=25)`
12. Register connection: `_ws_register(conn_id, sock, ping_thread)`
13. Enter blocking WebSocket echo loop (scaffold for Phase 28 voice relay):
    ```
    try:
        while True:
            opcode, payload = ws_recv_frame(sock)
            if opcode is None or opcode == WS_OP_CLOSE:
                break
            if opcode == WS_OP_PING:
                ws_send_frame(sock, WS_OP_PONG, payload)
                continue
            if opcode == WS_OP_PONG:
                continue
            # Echo text and binary frames back (Phase 28 replaces with Gemini relay)
            if opcode in (WS_OP_TEXT, WS_OP_BINARY):
                ws_send_frame(sock, opcode, payload)
    except (ConnectionError, OSError, socket.timeout):
        pass
    finally:
        _ws_unregister(conn_id)
        try:
            ws_send_close(sock)
        except Exception:
            pass
        _ws_log.info("WebSocket closed", extra={"event": "ws_close", "conn_id": conn_id})
    ```
